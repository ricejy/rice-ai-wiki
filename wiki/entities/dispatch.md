# Dispatch

Dispatch is a feature within [[Anthropic]]'s Claude Cowork enterprise workspace that allows users to assign tasks to agents via text or voice from anywhere in the application. It acts as the universal task entry point across the Cowork ecosystem, including [[Model Context Protocol]]-connected tools and Microsoft 365 integrations.

## Key characteristics

- Voice and text input for task assignment
- Works across all Cowork-connected applications (Excel, PowerPoint, Word, Outlook)
- Context carries between applications — tasks started in one tool don't need re-explanation when moving to another
