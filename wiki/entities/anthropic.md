# Anthropic

## Overview

AI safety company building the Claude model family. Known for emphasis on safety research alongside capability development.

## Key products and models

- **Claude** — foundation model family. Opus 4.7 leads Vals AI Finance Agent benchmark (64.37%)
- **[[Claude Code]]** — agentic coding CLI/IDE tool
- **Claude Cowork** — enterprise agent workspace with plugin system and [[Dispatch]] (voice/text task assignment)
- **[[Claude Managed Agents]]** — autonomous agent platform with audit logging, per-tool permissions, credential vaults
- **[[Claude Agent SDK]]** — SDK for building agent applications
- **[[Model Context Protocol]]** — standard protocol for connecting LLMs to tools and data sources
- **[[Sonnet 4.6]]** — used as the classifier model in Claude Code auto mode
- **Microsoft 365 add-ins** — Claude in Excel, PowerPoint, Word, Outlook with cross-app context

## Enterprise: financial services

10 agent templates for finance (pitchbooks, KYC, month-end close). Connector ecosystem with FactSet, S&P, Moody's, and dozens more. See [[Agents for Financial Services]].
