# Claude Agent SDK

## Overview

[[Anthropic]]'s SDK for building agent applications on top of Claude. One of several frameworks mentioned in "Building Effective Agents," alongside AWS Strands Agents SDK, Rivet, and Vellum.

[[Anthropic]]'s own guidance: start with direct API calls before reaching for the SDK. Frameworks add useful abstraction but can obscure underlying logic and complicate debugging. Use the SDK once [[Agent Design Patterns]] are proven and the application warrants the abstraction.
