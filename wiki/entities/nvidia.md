# Nvidia

## Overview

GPU manufacturer and AI infrastructure company. Beyond hardware, contributes significant research to inference optimization and model efficiency.

## Key contributions (in wiki)

- **Dynamic Memory Sparsification (DMS)** — retrofittable [[Sparse Attention]] method with delayed eviction. Up to 8x reasoning cost reduction on some models.
- **KV Cache Transform Coding (KVTC)** — PCA-based [[KV Cache]] compression, analogous to JPEG. Up to 20x memory reduction and 8x TTFT improvement without model retraining.
